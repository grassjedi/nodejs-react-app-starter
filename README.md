# Starter Web App

This is my attempt to create a dev workflow which hot reloads client and server side changes.

Note: This is still a work in progress

## Installation

    git clone xyz
    rm -rf .git
    npm install
    gulp (npm run dev)
    open http://localhost:3000

## Deployment

    gulp bundle (npm run bundle)
